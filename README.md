# Cyber Shield Student

Cyber Shield – Student Security App"

Features:

Password strength checker

Phishing detector

Security tips

Data breach alert

Quiz system

## Getting Started

First, install dependencies:

```bash
npm install
# or
pnpm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
# or
pnpm dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
